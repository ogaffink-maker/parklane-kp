PARK LANE × GASTRO ULTIMA — медиа для сайта
=============================================

Клади файлы прямо в эту папку (assets/) с ТОЧНО такими именами.
Сайт сам подхватит их. Если файла нет — на месте кадра останется
рамка-заглушка с подписью, ничего не сломается.

ВИДЕО ТОЛЬКО ОДНО — hero.mp4 (уже на месте). Оно привязано к скроллу:
прокрутка страницы прокручивает ролик по кадрам, видно целиком.

Остальные кадры — ФОТО:

ИМЯ ФАЙЛА      ГДЕ НА САЙТЕ             ЧТО ПОЛОЖИТЬ
-------------  -----------------------  ----------------------------------------
hero.mp4       Открывающий кадр         УЖЕ ЕСТЬ (видео, скрабится скроллом)
chef.jpg       Портрет в досье шефа     УЖЕ ЕСТЬ (вертикальное, 3:4)
kitchen.jpg    После досье шефа         Кухня: огонь, плита, руки шефа
plating.jpg    После списка услуг       Подача блюд, сервировка, новое меню
finale.jpg     Финал (после контактов)  Зал Park Lane в полном свете

ТРЕБОВАНИЯ К ФОТО:
- JPG, ширина 1600–2000px достаточно
- Горизонтальные (16:9 или 3:2) — рамка сама подстроится под пропорции
- Тёмные/вечерние кадры смотрятся лучше всего

После добавления файлов — обновить сайт:
  git add assets
  git commit -m "фото"
  git push
Через минуту обновится https://ogaffink-maker.github.io/parklane-kp/
